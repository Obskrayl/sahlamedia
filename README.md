# Sahla Media

موقع شركة سهلة ميديا (Sahla Media) للتسويق الرقمي وحلول الويب.

## عن المشروع
هذا المشروع عبارة عن صفحة هبوط (Landing Page) لشركة تسويق رقمي، تم بناؤها باستخدام React وتتميز بتصميم عصري ومتجاوب.

## التقنيات المستخدمة
- **React.js**: لبناء واجهة المستخدم.
- **Tailwind CSS**: لتنسيق العناصر وتصميمها.
- **Lucide Icons**: للأيقونات.
- **Tajawal Font**: الخط العربي المستخدم.

## المميزات
- تصميم متجاوب (Responsive Design).
- تأثيرات حركية (Animations) جذابة.
- الوضع الليلي (Dark Theme) كنمط أساسي.
